⚙️ パラメータ詳細

主要パラメータ

  • font: フォント名（Inter, Arial, Fira+Code, Noto+Sans+JP など）
  • size: フォントサイズ（数値）
  • color: テキストカラー（16進数、色名）
  • width: SVGの幅（ピクセル）
  • height: SVGの高さ（ピクセル）
  • lines: 表示するテキスト（セミコロン区切り）
  • center: 水平中央揃え（true/false）
  • vCenter: 垂直中央揃え（true/false）
  • pause: 各行の間隔（ミリ秒）
  • duration: タイピング速度（ミリ秒）
  • repeat: リピート（true/false）

💡 実装のヒント

  • レスポンシブ対応: max-width: 100% を CSS で設定し、画面サイズに応じてフォントサイズを調整
  • アクセシビリティ: 必ず alt 属性を設定し、スクリーンリーダー対応を行う
  • パフォーマンス: 必要以上に長いテキストは避け、適切なキャッシュ設定を行う
  • SEO対策: 重要なテキストは通常のHTMLでも提供し、検索エンジンが認識できるようにする
  • フォールバック: 画像が読み込まれない場合の代替テキストを準備し、ユーザビリティを確保
  • モバイル最適化: タッチデバイスでの操作性を考慮し、適切なサイズ設定を行う
  • 読み込み速度: 外部サービス利用時は読み込み時間を考慮した設計を行う

const TypingHeader = ({ lines, color = "4F46E5", size = 24, font = "Noto+Sans+JP" }) => {
  const typingUrl = `https://readme-typing-svg.demolab.com?font=${font}&size=${size}&color=${color}¢er=true&vCenter=true&width=600&lines=${lines.join(';')}`;
  
  return (

        src={typingUrl}

};
